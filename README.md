# Shape
The project uses only the C++ standard library and is easy to compile on any modern compiler.

The game revolves around identifying geometric patterns represented as a circle, triangle, and square. Each round presents a random shape challenge, and the player must choose the correct answer to gain points. The simplicity of the design makes it ideal for learning loops, conditionals, and random number generation.

The structure is intentionally minimal so developers can easily extend it with new shapes, scoring systems, or difficulty levels. It serves as a foundation for experimenting.
